# Senior-Design: IED Detection Pipeline

## Project Overview
This project aims to develop a machine learning pipeline to detect interictal epileptiform discharges (IEDs) in intracranial electroencephalography (iEEG) data.

## How to Run
### Create and activate a virtual environment
python3 -m venv .venv
source .venv/bin/activate

### Install dependencies
pip install -r requirements.txt
